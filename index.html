Maestro Cup ADMIN

Panel de gestion du tournoi

Connexion admin

Utilise le compte créé dans Firebase Authentication.

Sélectionne les matchs actuellement en cours. Un match "en direct" apparaît en tête du site public.

Applique une pénalité (points retirés au classement de poule) à une équipe.

Les équipes qualifiées apparaissent automatiquement une fois les poules terminées. Renseigne juste les scores.

Initialiser les données du tournoi

À utiliser une seule fois, avant le début du tournoi : crée les 16 équipes et les 24 matchs de poule dans Firestore. Si des données existent déjà, cette action ne fera rien pour éviter d'écraser des scores en cours.
